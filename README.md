# Cluboard

**What's in the Cluboard?** A mobile-first school club discovery app with an Instagram-inspired UI.

Find what's inside your school's clubs — browse the directory, view club profiles, and see recent updates. No login required for the MVP; anyone can create or edit club pages.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser. For the best experience, use mobile viewport or device emulation in DevTools.

## Features

- **Home / Directory** — Searchable club list with featured row, filters, and recent updates feed
- **Club Profile** — Instagram-style layout with About, Schedule, Posts, and Contact tabs
- **Create / Edit Club** — Simple forms for all club fields
- **Explore** — Search, filter, and sort clubs
- **Seed data** — National Honor Society and Interact preloaded

## Tech stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
- React Router 7
- localStorage persistence
